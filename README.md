# powerClipboard
静默式自动翻译,自动OCR,自动公式识别

初次启动较慢,后期考虑使用GoLang重构

在`./src/pic2text.py`内填上你的百度OCR API即可,或者使用我提供的二进制包.
## 使用方法
```
git clone https://github.com/trotsky1997/powerClipboard.git
cd powerClipboard
python app.py
```
